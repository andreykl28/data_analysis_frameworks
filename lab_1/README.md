# Лабораторная работа №1

Лабораторная работа по базовому ML-пайплайну на табличных данных.
В работе используется датасет `student_lifestyle_100k.csv` с информацией о режиме дня, учебной нагрузке, активности, стрессе, успеваемости и признаке депрессии у студентов.

## Что сделано

В ноутбуке выполнен разведочный анализ данных: просмотр структуры датасета, описательных статистик, распределений, выбросов и корреляций.

На этапе предобработки удален неинформативный `Student_ID`, убраны строки с некорректной суммой часов в сутках и дубликаты, категориальные признаки закодированы в числовой вид. Также добавлены простые признаки `Free_Time` и `Activity_Hours`.

Данные разделены на обучающую, валидационную и тестовую выборки. Масштабирование выполняется после разделения, чтобы не было утечки данных.

Для регрессии обучена `LinearRegression`, которая прогнозирует `CGPA`. Качество оценивается по MAE, MSE, RMSE и R².

Для классификации обучена логистическая регрессия через `SGDClassifier(loss="log_loss")`, чтобы можно было менять learning rate и число эпох. Качество оценивается по Accuracy, Precision, Recall, F1 и ROC-AUC, отдельно проверяется отсутствие выраженного переобучения по train/validation/test.

## Файлы

- `notebook.ipynb` - основной ноутбук с анализом данных.
- `data/student_lifestyle_100k.csv` - датасет.

## Требования

- Python 3.10 или новее.
- Доступ в интернет для первичной загрузки датасета через `kagglehub`, если он не был скачан ранее.

## Запуск

1. Перейти в папку проекта:

```bash
cd lab_1
```

2. Создать и активировать виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate
```

Для Windows:

```bash
.venv\Scripts\activate
```

3. Установить зависимости:

```bash
python -m pip install --upgrade pip
python -m pip install pandas numpy seaborn matplotlib kagglehub ipykernel jupyter scikit-learn
```

4. Добавить виртуальное окружение как kernel для Jupyter:

```bash
python -m ipykernel install --user --name lab_1 --display-name "Python (lab_1)"
```

5. Запустить Jupyter Notebook:

```bash
jupyter notebook
```

6. Открыть файл `notebook.ipynb`, выбрать kernel `Python (lab_1)` и выполнить все ячейки через `Run All`.

## Примечание

Если при выполнении первой ячейки появляется ошибка `ModuleNotFoundError`, нужно убедиться, что выбран kernel `Python (lab_1)`, созданный из виртуального окружения `.venv`.
